Resumen del Proyecto

"El Huesesaso" es un dashboard web interactivo diseñado como prototipo para un concesionario de vehículos de lujo. La plataforma, construida con HTML y CSS, presenta un diseño moderno y responsivo que se adapta a diferentes dispositivos.

Funcionalidades principales:

  • Dashboard Principal: Muestra los logotipos de las marcas de vehículos y una selección de coches recomendados.
  • Páginas de Marcas: Secciones dedicadas para Lamborghini, Mercedes-Benz, BMW y Porsche donde se listan los modelos disponibles.
  • Detalles del Vehículo: Al hacer clic en un vehículo, el usuario es dirigido a una página con una descripción detallada, especificaciones técnicas y características destacadas.
  • Servicios de Mantenimiento: Una sección informativa que lista los centros de servicio autorizados para cada marca en Colombia, con enlaces a Google Maps para una fácil ubicación.

El objetivo del proyecto es demostrar la creación de una interfaz de usuario atractiva y funcional para la industria automotriz de lujo, enfocándose en la experiencia de usuario y la presentación de la información de manera clara y elegante.
